Endure

Procedurally generated two dimensional action and adventure game made in a roguelike fashion.

About The Game

Do you like roguelike games? Have you ever wanted to play on a completly randomized map every time? If you answered yes to those two questions then Endure™ is the game for you. Wander the wilds of a post apocolyptic world, fighting to stay alive in an unforgiving and hostile land. Solve challenges and figure out how to make it to your objective in one piece.

We are providing this game free and also making it open source. The idea is to have an entire community help develop and make this game amazing. If you would like to contribute to this game please review the readme on our github site and fork the repo when you are ready. Also note that any proceedes generated from this game will go to charity.

This project was created with Unity, Unity3d and C#
